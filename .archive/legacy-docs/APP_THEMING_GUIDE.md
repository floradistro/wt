## üì± App Icon Generation

### Requirements

Your app icon should be:
- **Size**: 1024x1024px (required for App Store)
- **Format**: PNG with transparency
- **Background**: Can use Apple liquid glass effect
- **Logo**: Centered Whaletools logo

### Option 1: Manual Design (Recommended)

Use Figma, Sketch, or Photoshop to create:

1. **Create base** (1024x1024px):
   - Background: Black (#000000) or dark gradient
   - Apply subtle glass/blur effect
   - Border: Optional 1px white at 15% opacity

2. **Add logo**:
   - Use `assets/logo.png` as source
   - Center it in the canvas
   - Size: ~60% of total canvas (leave breathing room)
   - Apply subtle drop shadow for depth

3. **Apple Liquid Glass Effect**:
   - Add frosted glass overlay (white at 5-8% opacity)
   - Apply subtle radial gradient for depth
   - Optional: Add subtle inner glow on edges

4. **Export settings**:
   - Format: PNG
   - Color profile: sRGB
   - Resolution: 1024x1024px @1x

### Option 2: Use Icon Generator Service

Use [Expo's icon generator](https://icon.kitchen) or similar:

1. Upload your logo (`assets/logo.png`)
2. Choose:
   - Background: Black
   - Padding: 20%
   - Style: iOS (rounded square)
3. Download all sizes
4. Replace files in `assets/` folder

### Icon Sizes Required

After creating your 1024x1024 master icon, you need these sizes:

#### iOS
- `icon.png`: 1024x1024 (App Store)
- Auto-generated by Expo for all device sizes

#### Android
- `adaptive-icon.png`: 1024x1024 (foreground)
- Background color: #000000 (set in app.json)

### File Locations

Replace these files:

## üåä Splash Screen Design

### Current Configuration

```json
"splash": {
  "image": "./assets/splash-icon.png",
  "resizeMode": "contain",
  "backgroundColor": "#000000"
}
```

### Splash Screen Image

Create `splash-icon.png` (1024x1024px):

1. **Background**: Transparent PNG
2. **Logo**: Centered Whaletools logo
3. **Size**: Logo should be ~40% of canvas
4. **Style**: Clean, minimal - logo only

The splash screen will display:
- Black background (from backgroundColor)
- Your logo centered (from splash-icon.png)
- Smooth transition to LoadingScreen

### Design Tips

- Keep it simple - just logo, no text
- Ensure logo is visible on black background
- Use high contrast for quick recognition
- Logo should match app icon design language

## üéØ Design Guidelines

### Apple Standards

Follow Apple's Human Interface Guidelines:

1. **App Icon**:
   - No transparency (except adaptive icon foreground)
   - Simple, recognizable design
   - Consistent with brand identity
   - Avoid text (use symbol/logo only)

2. **Color Scheme**:
   - Pure black background (#000000)
   - White/light elements for contrast
   - Subtle opacity for glass effects
   - Maintain visual hierarchy

3. **Launch Experience**:
   - Splash ‚Üí Loading ‚Üí Login (smooth transitions)
   - No jarring color changes
   - Consistent dark theme throughout
   - Haptic feedback on interactions

### Brand Consistency

Ensure all visual elements match:
- Logo usage (consistent across icon, splash, loading)
- Color palette (black, white, subtle grays)
- Typography (SF Pro on iOS, Roboto on Android)
- Glass/blur effects (liquid glass throughout)

## üêõ Troubleshooting

### Icon not updating
```bash
# Clear Expo cache
npx expo start --clear

# Clear iOS derived data
rm -rf ~/Library/Developer/Xcode/DerivedData

# Rebuild
npx expo prebuild --clean
```

### Splash screen shows white background
- Check `app.json` splash backgroundColor is "#000000"
- Verify splash-icon.png has transparent background
- Clear cache and rebuild

### Loading screen not showing
- Check LoadingScreen.tsx import path
- Verify isRestoringSession state logic
- Check ErrorBoundary isn't catching errors
